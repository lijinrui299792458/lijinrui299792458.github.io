## 跳跃游戏1

给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个位置。

示例：

输入: [2,3,1,1,4]
输出: true
解释: 我们可以先跳 1 步，从位置 0 到达 位置 1, 然后再从位置 1 跳 3 步到达最后一个位置。

解答：

其实一个简单的思路就是保存跳跃过程中最远能跳的节点，若最远节点大于最后一个节点则一定能跳跃成功；如果跳跃过程中出现最远节点比当前节点要小，说明连当前这个节点都跳不到，返回false。

```C++
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int max_jump = 0;
        for (int i = 0; i < nums.size(); ++i) {
            if (i > max_jump) return false;
            max_jump = max(max_jump, i + nums[i]);
            if (max_jump > nums.size() - 1) return true;
        }
        return true;
    }
};
```

## 跳跃游戏II

给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

你的目标是使用最少的跳跃次数到达数组的最后一个位置。

示例:

输入: [2,3,1,1,4]
输出: 2
解释: 跳到最后一个位置的最小跳跃数是 2。
     从下标为 0 跳到下标为 1 的位置，跳 1 步，然后跳 3 步到达数组的最后一个位置。
说明:

假设你总是可以到达数组的最后一个位置。

解答：

**回溯法 or dp法 or 贪心法**

第一个想到的就是dfs：

```C++
int dp[100010];
class Solution {
public:
    int res = INT_MAX;
    void dfs(int index, vector<int>& nums, int cnt) {
        if (index >= nums.size() - 1) {
            res = min(res, cnt);
            return;
        }
        int step = nums[index];
        for (int i = 1; i <= step; ++i) {
            dfs(index + i, nums, cnt + 1);
        }
    }
    int jump(vector<int>& nums) {
        dfs(0, nums, 0);
        return res;
    }
};
```

但是这样做会有很大的开销，时间复杂度在指数级别，会TLE。

再看第二个dp方法：

- 初始化状态:因为涉及到比较,因此对所有的下标位设置一个不可能到达的值 nums.size() + 1
- 状态转移方程: dp[i] = min(dp[j]+1,dp[i]); dp[i]表示走到当前位置最少需要的步骤 

```C++
int dp[100010];
class Solution {
public:
    int jump(vector<int>& nums) {
        for (int i = 0; i < nums.size(); ++i) dp[i] = nums.size();
        dp[0] = 0;
        for (int i = 1; i < nums.size(); ++i) {
            for (int j = 0; j < i; ++j) {
                if (j + nums[j] >= i)dp[i] = min(dp[j] + 1, dp[i]);
            }
        }
        return dp[nums.size() - 1];
    }
};
```

原理大致是这样的，dp[i]表示当前位置最少的步骤数，所以我们可以看在[0, i-1]这个区间里面是否有`nums[j] + j >= i`的值，如果有的话说明从j就能一步跳到位置i上，我们只需要保存从[0, i-1]这个区间里面最少的步骤数就可以了。但是此法仍然会TLE。

而贪心法其实是本题的最优解法：

```C++
int jump(vector<int> &nums)
{
    int ans = 0;
    int start = 0;
    int end = 1;
    while (end < nums.size())
    {
        int maxPos = 0;
        for (int i = start; i < end; i++)
        {
            // 能跳到最远的距离
            maxPos = max(maxPos, i + nums[i]);
        }
        start = end;      // 下一次起跳点范围开始的格子
        end = maxPos + 1; // 下一次起跳点范围结束的格子
        ans++;            // 跳跃次数
    }
    return ans;
}
```

找到每一步里能跳最远的，并且在每一步都更新起点和终点。

优化

- 从上面代码观察发现，其实被 while 包含的 for 循环中，i 是从头跑到尾的。
- 只需要在一次 跳跃 完成时，更新下一次 能跳到最远的距离。
- 并以此刻作为时机来更新 跳跃 次数。
- 就可以在一次 for 循环中处理。

```C++
int jump(vector<int>& nums)
{
    int ans = 0;
    int end = 0;
    int maxPos = 0;
    for (int i = 0; i < nums.size() - 1; i++)
    {
        maxPos = max(nums[i] + i, maxPos);
        if (i == end)
        {
            end = maxPos;
            ans++;
        }
    }
    return ans;
}
```
