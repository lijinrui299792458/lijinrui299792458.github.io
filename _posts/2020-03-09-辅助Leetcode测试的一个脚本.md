> 辅助Leetcode测试的一个自用脚本

在做leetcode的时候，题目的输入格式通常是：[1, null, 2, 3...]一类的。而在C++代码中是使用`vector<int> t = {1, 2, 3...}`进行表示的，就为编写代码写测试用例的时候造成了很大的困扰。所以如果有一种脚本可以根据[1,null,2,3...]直接生成相应的数据结构，再结合算法就好了。所以自己写了一个脚本：HelpLeetcode.h , 目前只实现了树结构，但是后面会陆续加入新的数据结构和更多的复用接口。脚本代码参见：[https://github.com/YunLambert/SomeScripts/blob/master/HelpLeetcode/HelpLeetcode.h](https://github.com/YunLambert/SomeScripts/blob/master/HelpLeetcode/HelpLeetcode.h)

## 接口

- [x] 树
- [ ] 链表
- [ ] 图
- [ ] 二维数组（矩阵）

## 构建一棵树

接口目前为：

构建一棵树，返回根节点：

```C++
TreeNode* hlc::makeTreeTest(string& s);
```

打印整棵树：

```C++
Print(TreeNode* root);
```

```
 /*
     * Make a tree according with string s.
     * Example:
     * Input: "[1, null, 2, 3, 4]"
     * Output: TreeNode*  1
     *                   / \
     *                 null 2
     *                     /  \
     *                    3    4
     * */
```

```C++
TreeNode* makeTreeTest(std::string& s) {
  std::vector<int> nums;
  nums = parseString<int>(s);
  size_t count = nums.size();
  TreeNode **treeArr = new TreeNode*[count];
  for (size_t i = 0; i < count; i++)
  {
    if (INT_MAX == nums[i]) {
      treeArr[i] = NULL;
    } else {
      treeArr[i] = new TreeNode(nums[i]);
    }
  }

  size_t curr = 1;
  for (size_t i = 0; i<count; i++)
  {
    if( !treeArr[i] ) {
      continue;
    }

    if (curr < count) {
      treeArr[i]->left = treeArr[curr++];
    }
    if (curr < count) {
      treeArr[i]->right = treeArr[curr++];
    }
  }

  TreeNode *root = treeArr[0];
  delete[] treeArr;
  return root;
}
```

TreeNode* makeTreeTest2(std::string& s) {
  std::vector<int> nums;
  std::vector<TreeNode*> t;

  nums = parseString<int>(s);
  for (auto it : nums) {
    if (it == INT_MAX) t.push_back(NULL);
    else {
      TreeNode *node = new TreeNode(it);
      t.push_back(node);
    }
  }
  if (t.empty()) return NULL;
  int index = 1;
  while (index*2 + 1 <= t.size()) {
    if (index*2 <= t.size() && t[index*2 - 1] != NULL) t[index - 1]->left = t[index*2 - 1];
    if (index*2 + 1 <= t.size() && t[index*2] != NULL) t[index - 1]->right = t[index*2];
    ++index;
  }
  TreeNode* testNode = t[0];
  return testNode;
}
