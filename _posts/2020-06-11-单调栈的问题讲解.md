> 单调栈问题讲解

单调栈就是严格单调递增或单调递减的栈。使用的场景：**利用单调栈，可以找到从左/右遍历第一个比它小/大的元素的位置**

下面就是一个简单的单调栈的模板题：

给定一个长度为N的整数数列，输出每个数左边第一个比它小的数，如果不存在则输出-1。

例如输入为：

```
5
3 4 2 7 5
```

则输出就为：

```
-1 3 -1 2 2
```

代码这样实现：

```C++
#include <iostream>
#include <stack>
using namespace std;
stack<int> s;
int main() {
    int n;
    cin>>n;
    while (n--) {
        int x;
        cin>>x;
        while (!s.empty() && s.top() >= x) s.pop();
        if (!s.empty()) cout<<s.top()<<" ";
        else cout<<"-1 ";
        s.push(x);
    }
    return 0;
}
```

如何判断什么时候该入栈还是出栈，栈顶元素和数组元素( s.top() 和 x )之间是大于等于还是小于等于需要结合具体问题进行分析；我个人的经验是要求哪边最近，就从那边开始扫描。比如说求左边临近且比它小的元素，就从左边开始扫描；求右边临近比它大的元素，就从右边开始扫描。然后关于最小最大，例如求2 1 2 4 3的右边临近且比它大的元素，就想象成有5个人，身高分别为2 1 2 4 3，**他们看到人的标准是仰头到平视过程中的第一人**，2向右看第一个看到的只能是4，1比2矮所以看不见；1向右看只能看到2.....所以从右边开始扫描，如果当前元素的身高比栈内某些元素都高，那么当前元素能够看见的应该是栈内比它高的元素，举个例子：4，3，5，如果栈内当前是[3,5]，到4的时候4只能看见5，而这个看见就是我们需要找的栈顶元素，在5之前的元素都应该pop出去。如何维护pop的条件，就是(s.top() <= x)))

再举个例子，求3 4 2 7 5的左边邻接且比它小的元素，身高为3 4 2 7 5的人，**他们看到人的标准是低头到平视过程中看见的第一人**，4向左低头能看见的第一人为3，7向左低头能看见的第一人为2。所以从左边开始扫描，当前元素能够看见的应该是栈内最矮的人，举个例子1,3,5,2，如果栈内当前是[1,3]，那么5只能看见3，2能看见的第一人是1，那么在1之后的元素应该都pop出去。如何维护pop的条件，就是(s.top() >= x)))

仰头到平视过程就是单调增栈，就是比我矮的都从栈里出去；低头到平视过程就是单调减栈，就是比我高的都从栈里出去。

### Leetcode 739

[https://leetcode-cn.com/problems/daily-temperatures/](https://leetcode-cn.com/problems/daily-temperatures/)

```C++
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& T) {
        stack<pair<int, int>> s;
        vector<int> res;
        for (int i = T.size() - 1; i >= 0; --i) {
            while (!s.empty() && s.top().first <= T[i]) s.pop();
            if (!s.empty()) res.push_back(s.top().second - i);
            else res.push_back(0);

            s.push({T[i], i});
        }
        reverse(res.begin(), res.end());
        return res;
    }
};
```
