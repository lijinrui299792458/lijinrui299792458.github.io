> 反转链表的递归解法

题目是Leetcode的203题：[https://leetcode-cn.com/problems/reverse-linked-list/](https://leetcode-cn.com/problems/reverse-linked-list/)

链表题本身就很巧妙，关于这道题也有很多解法，指针指来指去的，在这里首先给一个自己最常用的写法：

```C++
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* p = nullptr, *cur = head;
        while (cur) {
            ListNode* tmp = cur->next;
            cur->next = p;
            p = cur;
            cur = tmp;
        }
        return p;
    }
};
```

但是最近在写递归解法的时候却卡住了，在这里总结一下：

递归解法I：

```C++
class Solution {
public:
    ListNode* reverseListCur(ListNode* node, ListNode* newHead) {
        if (!node) return newHead;
        ListNode* p = node->next;
        node->next = newHead;
        return reverseListCur(p, node);
    }
    ListNode* reverseList(ListNode* head) {
        return reverseListCur(head, nullptr);
    }
};
```

递归解法II:

```C++
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        if (!head || !head->next) return head;
        ListNode *p = reverseList(head->next);
        head->next->next = head;
        head->next = NULL;
        return p;
    }
};
```

注意这里两个解法的区别，解法2中是首先找到最后一个节点，然后在返回过程中进行reverse操作。而解法1中则是先reverse并将newHead更新成reverse后的头结点，最后直接返回newHead。相比而言，解法1在过程中newHead是不断更新的，解法2则是直接先找到newHead。
