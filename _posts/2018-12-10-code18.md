## 前言

当我们使用stl的时候，我们会遇到一种情况:如何在`std::vector`与`std::map`中获取每一项的第一个元素，对于map来说就是如何获得所有的key元素；虽然这个问题不复杂，但是这种情况出现频率还是挺高的，所以拿出来探究一下。
## 常规做法

```C++
#include<iostream>
#include<map>
#include<vector>
using namespace std;

template<typename T,typename S>
std::vector<T> get_first(map<T,S> a) {
	vector<T> tmp;
	for (auto it=a.begin(); it!=a.end(); ++it)
	{
		tmp.push_back(it->first);
	}
	return tmp;
}

int main()
{
	map<char, int> a = { {'a',1},{'d',2},{'g',3} };
	vector<char> res = get_first(a);
	for (auto c : res)
	{
		cout << c << " ";
	}
	cout << endl;
	system("pause");
}
```

## C++14中

在C++14中，我们可以用泛型lambada做到：

```C++
std::map<int, std::string> numbers_with_names = { {1, "one"}, {2, "two"}, {3, "three"} };
std::vector<int> numbers;
 
std::transform(begin(numbers_with_names),end(numbers_with_names),std::back_inserter(numbers),[](auto const& pair){ return pair.first; });
```

所谓泛型lambda就是在形参声明中使用auto类型指示说明符的lambda。

我们可以定义这一系列的东西到`namespace`中，然后在其他文件中就可以通过namespace访问到了，在这里我们用的是`const &`即`passed by reference`:

```C++
namespace util
{
    const auto get_first = [](auto const& pair) -> auto const& { return pair.first; };
}
```

## C++20的Range-V3库





```C++
struct get_first
{
    template<typename T, typename U>
    T const& operator()(std::pair<T, U> const& pair)
    {
        return pair.first;
    }
};
```
