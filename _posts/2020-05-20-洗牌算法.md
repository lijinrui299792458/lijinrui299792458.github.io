> 洗牌算法

打过扑克牌的人知道，一组新的牌往往很难“完全打乱”，在玩了几局后才会觉得比较乱了。在程序实现时如何去定义一个“乱”，怎样体现这种随机性？这种算法一般称为“洗牌算法”。

洗牌算法是将原来的数组打散，使原数组的某个数在打散后的数组中的每个位置上等概率的出现。

## Fisher-Yates Shuffle算法

从原始数组中随机选一个之前没取过的数字到新的数组中，程序实现如下：

```C++
#include <iostream>
#include <ctime>
#include <vector>
using namespace std;
const int M = 5;
void shuffle(vector<int>& arr, vector<int>& newarr) {
    srand((unsigned)time(NULL));
    for (int i = 0; i < M; ++i) {
        int k = rand()%arr.size();
        newarr.push_back(arr[k]);
        arr.erase(arr.begin() + k);
    }
}
```

基本思路就是将一个数组arr乱序的结果存放到newarr里，每次在arr中随机选择一个元素放入newarr然后删除arr中的该元素，下一次循环中在剩下的arr中再次重复以上过程。每一次随机的范围都在慢慢变小。

证明随机性也很简单：新数组中每个新元素的概率为$P=\frac{n-1}{n}*...\frac{1}{n-i+1}=\frac{1}{n}$

## Knuth-Durstenfled Shuffle算法

之前Fisher的算法需要$O(n)$的额外空间复杂度，如果在原始数组上直接对元素进行交换则可以省去这部分空间，实现代码如下：

```C++
void shuffle(vector<int>& arr) {
    for (int i = arr.size() - 1; i >= 0; --i) {
        srand((unsigned)time(NULL));
        std::swap(arr[rand()%(i + 1)], arr[i]);
    }
}
```

在这个算法中是将随机的数字放在数组尾部，是一种原地打乱算法。

验证随机性：

`arr[i]`这个元素通过交换一次后到`arr[n-1]`位置的概率是$\frac{1}{n}$，通过交换2次后到`arr[n-2]`位置的概率为$\frac{n-1}{n}*\frac{1}{n-1} = \frac{1}{n}$，以此类推，`arr[i]`这个元素交换i次到`arr[n-i]`位置的概率为$\frac{1}{n}$，由于i具有任意性，所以arr中的每个元素都可以确保达到新的位置，且概率为$\frac{1}{n}$

## 一些思考

最近突发奇想的搜索了洗牌算法，是因为有一个“题库出题”的需求，就是从一个题库里随机出10道题发给学生们测试，如果该学生这次未做对6题，则需要重新出10道题。而具体的任务就是让每次10道题都不同且每个学生得到的测试题目不同（可以有相同的题目，但是题目顺序最好不一致，防止作弊）。首先因为存储题库的数据库很大，如果学生每请求一次测试都需要重新排列数据库里的数据不太现实。

所以我的想法是维护一个出题列表和学生列表，首先将所有题目通过洗牌算法加入到题目队列中，假设所有题目共有90题，那么此时题目队列里就有90道题目，每个学生分配30道题目（10道题目为一次测试，三次测试都没过的话说明没通过测试！）所以题目队列里会根据学生测试次数为给学生推送剩下的题目数，如学生1参加过1次测试，我们只用给他再推送20个题目就行了。但是推送题目这个事件我觉得一次性会更好，一次分配30个题目。如果总共有4个学生，题目只有90道，则在3个学生分配完成后，第4个学生将重新执行shuffle进行分配工作。

```C++
#include <iostream>
#include <vector>
#include <queue>
using namespace std;
const int N = 90;  // total problems
const int M = 10;  // test student number
vector<int> problems_id(N, 0);
queue<int> st_list;
queue<int> p_list;

struct student {
    int id = -1;
    int arr[30] = {0};
    int cur = 0; // when cur equals to 3, means student fail the test
}st[M];

void add_problems_id(int id) {
    problems_id.push_back(id);
}

void remove_problems_id(int id) {
    auto it = find(problems_id.begin(), problems_id.end(), id);
    if (it != problems_id.end()) problems_id.erase(it);
}

void shuffle(vector<int> arr, queue<int>& newarr) {
    srand((unsigned)time(NULL));
    for (int i = 0; i < N; ++i) {
        int k = rand()%arr.size();
        newarr.push(arr[k]);
        arr.erase(arr.begin() + k);
    }
}

void Init() {
    for (int i = 1; i<=300; ++i) problems_id[i] = i;
    st[15].id = 15, st[16].id = 16, st[17].id = 17, st[18].id = 18;
    st[15].cur = 1, st[16].cur = 2, st[17].cur = 3, st[18].cur = 0;
    st_list.push(st[15].id);
    st_list.push(st[16].id);
    st_list.push(st[17].id);
    st_list.push(st[18].id);
}

void printStuProblem(int st_id) {
    int *arr = st[st_id].arr;
    int cur = st[st_id].cur;
    if (cur == 3) cout<<"Student"<<st_id<<" doesn't pass the test!";
    else for (int i = cur*10; i < 30; ++i) {
        cout<<arr[i]<<" ";
    }
    cout<<endl;
}


int main() {
    Init();
    shuffle(problems_id, p_list);
    while (!st_list.empty()) {
        cout<<st_list.size()<<" "<<p_list.size()<<endl;
        if (st_list.size()*30 > p_list.size()) {
            shuffle(problems_id, p_list);
        }

        int t = st_list.front();
        st_list.pop();
        if (st[t].cur == 3) continue;
        for (int i = 0; i < 30 - st[t].cur*10; ++i) {
            st[t].arr[i] = p_list.front();
            p_list.pop();
        }
    }

    cout<<"****TEST BEGIN*****"<<endl;
    cout<<"Student15 get 20"<<endl;
    printStuProblem(15);
    cout<<"Student16 get 10"<<endl;
    printStuProblem(16);
    cout<<"Student17 get 0"<<endl;
    printStuProblem(17);
    cout<<"Student18 get 30"<<endl;
    printStuProblem(18);

    return 0;
}
```
