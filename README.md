# YunLambert Github.io
利用hanuman的Jeklly模板进行搭建。

## 使用方法
1. 具体的网站信息都已经根据自己的需要进行了修改和完善。博客里的HOME和CONTACT会自动连接到我在博客园的网址和我的github。  
2. 发表文章的时候需要注意tags和发表文章的格式，发表文章的tags网站模板会自动进行分类，发文章只用在本地用HTML或者Markdown写好上传到_posts这个文件夹中就行了，命名格式为"datatime(%Y-%M-%D)-<子文件名称>"，如"2018-3-15-code"。  
3. ../assets/images里的图片的大小最好控制在200KB以下，否则网页加载的时候会比较慢。
