共和国春，中山园畔，鹤鸣九皋，声震于天。国运方兴，育才始建。开民智而传薪火，兴公学而启强源。四野子弟随军，八方才俊执鞭。琅琅书声晨曲，朦朦月色晚修。相帮教而学风善，兼体能而师道严。
峥嵘岁月，豪迈青年，凛凛傲骨，又红又专。神州惊雷彻响，辽沈优教初妍。人才早出，超常实验，优才之教，敢为人先。破常规而革羁绊，激潜能而求发展。
岁更千年，基教新篇。大学之道，兼容并建。一校主体多元，双制分轨运转。沧桑银杏不老，红砖古楼新颜。金发碧眼，相与同窗同友。日澳欧美，设帐学堂学院。招徕数理先锋，广纳杏坛诸贤。小儿游乐正酣，新疆葡萄又甜。蒲河岸边，细听莺燕频啼。丁香湖畔，惯看桃李芳芬。悲鸿丹青出彩，沈抚文化同城。
甲子重开，鼎新年代。因材施教，倾力创新。英才受教，开放拔尖。名师不凡，登临科研之巅。开实验班之首创，建实验室之尖端。名冠学界先导，誉满教改前沿。学子卓越，题名金榜之上，本土情怀淳厚，国际视野高远。出国门而境愈宽，入名校而意弥坚。健儿锦标夺魁，学子公益弘道。楫击鳞浪，琴奏殿堂。神州舞台出彩，联合国上建言。
岁月有痕，师道唯端，润物无声，大爱不言。耳萦教诲，力行弗忘，恩铭肺腑，感念衷肠。弘绩伟业，百年育才回望；展翔八荒，千载华夏畅想。
承父兄志而责成大业，系报国情而擎力担当。扫一屋以扫天下，成一事以成国家。励志笃学苦短，进德修身路长。“吾为中华之崛起而读书”，其言也铮铮乎凌越山川。“君子不匮，永锡尔类”，赤子拳拳，曦月煌煌。
